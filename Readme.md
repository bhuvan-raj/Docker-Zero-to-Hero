# 🐳 Docker — Zero to Hero

A structured, hands-on guide to learning Docker from the ground up — from understanding containers to deploying multi-container applications with Docker Compose.

> **GitHub:** [bhuvan-raj/Docker-Zero-to-Hero](https://github.com/bhuvan-raj/Docker-Zero-to-Hero)

---

## 📋 Prerequisites

Before you begin, make sure you have:

- A basic understanding of Linux command line
- Docker installed on your machine ([Installation guide →](01-introduction/README.md#-installing-docker))
- A [Docker Hub](https://hub.docker.com) account (required for the registry section)

## 🚀 How to Use This Guide

Each folder is self-contained. You can follow them in order for a complete learning path, or jump directly to the topic you need.

1. **Beginners** — Start from `01-introduction` and work through each section in order.
2. **Intermediate** — Jump to any specific topic using the table of contents above.
3. **Quick reference** — Each README is structured for easy scanning with clear sections and code blocks.
